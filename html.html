WHAT IS HTML?

Hypertext Markup Language (HTML) a standardized system for tagging text files to achieves on World Wide Web pages. a text-based approach to describing how content contained within an HTML file is structured. This markup tells a web browser how to display text, images and other forms of multimedia on a webpage.


• All HTML documents must start with a document type declaration: <!DOCTYPE html>

• The HTML document itself begins with <html> and ends with </html>.

• The visible part of the HTML document is between <body> and ends with </body>.

EXAMPLE :

  • <!DOCTYPE html>

  • <html>

  • <body>

  • <h1> My heading </h1>

  • <p> My paragraph </p>

  • </body>

  • </html>

What is <!DOCTYPE html>?

the <!DOCTYPE html> declaration represents the document type, and helps browsers to display web pages correctly.

It must only appear once, at the top of the page (before any HTML tags).

the <!DOCTYPE html> declaration is not case sensitive.

HEADING

h tag can be defined as a title or a subtitle which you want to display on the webpage. When you place the text within the heading tags, it is displayed on the browser in the bold format and size of the text depends on the number of heading.

HTML headings are defined with the <h1> to <h6> tags.

<h1> defines the most important heading. <h6> defines the least important heading: the h1 is the biggest and the h6 is to make the letters small.

EXAMPLE :

This is <h1>

This is <h2>

This is <h3>

This is <h4>

This is <h5>
This is <h6>

HTML PARAGRAPHS

To add a new paragraph in HTML, place your insertion mark cursor into the HTML document where you want to add the new paragraph

HTML paragraphs are defined with the <p> tag:

EXAMPLE :

<p>this a paragraph.</p>

<p>this is another paragraph.</p>

<p>. A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph..</p>

HTML LINKS

You can click on a link and jump to another document. When you move the mouse over a link, the mouse arrow will turn into a little hand. Note: A link does not have to be text. A link can be an image or any other HTML element!

HTML links are defined with the <a> tag.

EXAMPLE :

<a href="https://facebook.com">This is link</a>
